## 重ね着レベル

- `1` : 半袖シャツ１枚（夏とか）
- `2` : 半袖シャツ＋１枚 / 長袖シャツ１枚 (春から夏にかけて。夏から秋にかけて)
- `3` : ２枚（春や秋）
- `4` : 防寒肌着＋１枚＋コートやジャケット（冬〜春、秋〜冬）
- `5` : 防寒肌着＋２枚＋コート（真冬）
